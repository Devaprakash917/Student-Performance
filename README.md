This project performs Exploratory Data Analysis (EDA) on student performance data to understand the key factors influencing academic outcomes.

Using Python data analysis and visualization techniques, the project explores relationships between demographic, social, and academic variables to uncover meaningful insights that can help improve student performance and educational strategies.

Objectives:

Analyze student performance data

Identify factors affecting academic scores

Perform data cleaning and preprocessing

Visualize trends and relationships

Generate actionable insights

Key Insights:

Study time and preparation significantly influence scores

Parental education shows positive correlation with performance

Some demographic factors impact academic outcomes

Visualization helps identify performance patterns

Results:

The project demonstrates how exploratory data analysis can uncover meaningful patterns and relationships in educational datasets, helping stakeholders make informed decisions.
